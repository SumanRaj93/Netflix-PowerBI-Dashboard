# 🎬 Netflix Power BI Dashboard

An interactive Power BI dashboard built using the Netflix Titles dataset to explore content trends, genres, ratings, release years, and global distribution.


## 📌 Project Overview

This dashboard provides insights into Netflix's content library through interactive visualizations. The project demonstrates data cleaning, transformation, DAX, and dashboard design using Power BI.


## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel


## 📊 Dashboard Features

- 🎥 Movies vs TV Shows Distribution
- 🌍 Content by Country
- 🎭 Top Genres
- ⭐ Rating Distribution
- 🎬 Top Directors
- 📅 Content Added Over Time
- 📈 Release Year Trends
